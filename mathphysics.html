第一章：复变函数

问题1.1 - 柯西积分公式

题目描述

利用柯西积分公式计算积分：

\[ \oint_C \frac{e^z}{z^2 + 1} dz \]

其中C是正向圆周 \(|z| = 2\)。

解答

步骤1： 确定被积函数的奇点：

\[ z^2 + 1 = 0 \Rightarrow z = \pm i \]

这两个奇点都在圆周 \(|z| = 2\) 内。

步骤2： 应用柯西积分公式：

\[ \oint_C \frac{e^z}{z^2 + 1} dz = \oint_C \frac{e^z}{(z-i)(z+i)} dz \]

步骤3： 计算留数：

\[ \text{Res}(f, i) = \frac{e^i}{2i}, \quad \text{Res}(f, -i) = \frac{e^{-i}}{-2i} \]

步骤4： 应用留数定理：

\[ \oint_C = 2\pi i \left(\frac{e^i}{2i} + \frac{e^{-i}}{-2i}\right) = \pi(e^i - e^{-i}) = 2\pi i \sin(1) \]

第二章：积分变换

10题

问题2.1 - 傅里叶变换

题目描述

计算高斯函数 \( f(x) = e^{-ax^2} \) (a > 0) 的傅里叶变换。

解答

步骤1： 傅里叶变换定义：

\[ \hat{f}(k) = \int_{-\infty}^\infty e^{-ikx} e^{-ax^2} dx \]

步骤2： 配方：

\[ -ax^2 - ikx = -a\left(x + \frac{ik}{2a}\right)^2 - \frac{k^2}{4a} \]

步骤3： 变量替换并积分：

\[ \hat{f}(k) = e^{-\frac{k^2}{4a}} \int_{-\infty}^\infty e^{-a(x + \frac{ik}{2a})^2} dx = \sqrt{\frac{\pi}{a}} e^{-\frac{k^2}{4a}} \]
