第一章：微分几何基础

18题

问题1.1 - 克里斯托弗符号计算

题目描述

给定二维球面的度规：

\[ ds^2 = a^2(d\theta^2 + \sin^2\theta d\phi^2) \]

计算所有非零的克里斯托弗符号 \(\Gamma^\mu_{\nu\rho}\)。

解答

克里斯托弗符号的计算公式为：

\[ \Gamma^\mu_{\nu\rho} = \frac{1}{2}g^{\mu\sigma}(\partial_\nu g_{\rho\sigma} + \partial_\rho g_{\nu\sigma} - \partial_\sigma g_{\nu\rho}) \]

对于给定度规，非零分量为：

\[ g_{\theta\theta} = a^2, \quad g_{\phi\phi} = a^2\sin^2\theta, \quad g^{\theta\theta} = \frac{1}{a^2}, \quad g^{\phi\phi} = \frac{1}{a^2\sin^2\theta} \]

计算得到非零克里斯托弗符号：

\[ \Gamma^\theta_{\phi\phi} = -\sin\theta\cos\theta, \quad \Gamma^\phi_{\theta\phi} = \Gamma^\phi_{\phi\theta} = \cot\theta \]

第二章：爱因斯坦方程

15题

问题2.1 - 爱因斯坦张量计算

题目描述

证明爱因斯坦张量 \(G_{\mu\nu} = R_{\mu\nu} - \frac{1}{2}Rg_{\mu\nu}\) 满足 \(\nabla^\mu G_{\mu\nu} = 0\)。

解答

利用比安基恒等式：

\[ \nabla_\mu R^\mu_{\nu\rho\sigma} + \nabla_\rho R^\mu_{\nu\sigma\mu} + \nabla_\sigma R^\mu_{\nu\mu\rho} = 0 \]

收缩指标可得：

\[ \nabla^\mu R_{\mu\nu} - \frac{1}{2}\nabla_\nu R = 0 \]

这正是 \(\nabla^\mu G_{\mu\nu} = 0\)。
