第一章：自由场量子化

20题

问题1.1 - 克莱因-戈登场量子化

题目描述

推导自由实标量场(克莱因-戈登场)的正则量子化过程，包括场算符的展开和对易关系。

来源: Peskin《量子场论导论》第二章
解答

步骤1： 克莱因-戈登场的拉氏量密度：

\[ \mathcal{L} = \frac{1}{2}(\partial_\mu\phi)(\partial^\mu\phi) - \frac{1}{2}m^2\phi^2 \]

步骤2： 正则动量：

\[ \pi(x) = \frac{\partial\mathcal{L}}{\partial\dot{\phi}} = \dot{\phi}(x) \]

步骤3： 哈密顿量密度：

\[ \mathcal{H} = \pi\dot{\phi} - \mathcal{L} = \frac{1}{2}\pi^2 + \frac{1}{2}(\nabla\phi)^2 + \frac{1}{2}m^2\phi^2 \]

步骤4： 场算符的傅里叶展开：

\[ \phi(x) = \int \frac{d^3p}{(2\pi)^3}\frac{1}{\sqrt{2E_p}}(a_p e^{-ip\cdot x} + a_p^\dagger e^{ip\cdot x}) \]

步骤5： 对易关系：

\[ [a_p, a_{p'}^\dagger] = (2\pi)^3 \delta^{(3)}(\mathbf{p}-\mathbf{p}') \]

\[ [\phi(x), \pi(y)] = i\delta^{(3)}(\mathbf{x}-\mathbf{y}) \]

问题1.2 - 费米场量子化

题目描述

解释为什么费米场需要反对易关系而不是对易关系进行量子化。

解答

原因1： 自旋统计定理要求半整数自旋粒子服从费米-狄拉克统计。

原因2： 使用对易关系会导致负能态问题，违反正能量条件。

原因3： 反对易关系自然地实现了泡利不相容原理。

数学表达：

\[ \{\psi_a(x), \psi_b^\dagger(y)\} = \delta^{(3)}(\mathbf{x}-\mathbf{y})\delta_{ab} \]

其中 \(\{\cdot,\cdot\}\) 表示反对易子。

问题2.1 - φ⁴理论微扰展开

题目描述

计算φ⁴理论中两点关联函数的一阶微扰修正：

\[ \mathcal{L} = \frac{1}{2}(\partial_\mu\phi)^2 - \frac{1}{2}m^2\phi^2 - \frac{\lambda}{4!}\phi^4 \]

解答

步骤1： 相互作用项：

\[ H_{\text{int}} = \int d^3x \frac{\lambda}{4!}\phi^4 \]

步骤2： 一阶修正表达式：

\[ \langle \Omega|T\{\phi(x)\phi(y)\}|\Omega\rangle^{(1)} = -i\lambda \int d^4z D_F(x-z)D_F(z-z)D_F(z-y) \]

步骤3： 费曼图表示：

这是一个"蝌蚪图"贡献，包含一个自能圈。

步骤4： 动量空间表达式：

\[ \tilde{G}^{(1)}(p) = \tilde{D}_F(p) \cdot (-i\lambda) \left[\int \frac{d^4k}{(2\pi)^4} \frac{i}{k^2-m^2+i\epsilon}\right] \cdot \tilde{D}_F(p) \]

第三章：路径积分方法

15题

问题3.1 - 自由场生成泛函

题目描述

计算自由标量场的生成泛函 \(Z_0[J]\)，并推导两点关联函数。

解答

步骤1： 生成泛函定义：

\[ Z_0[J] = \int \mathcal{D}\phi \exp\left[i\int d^4x \left(\mathcal{L}_0 + J\phi\right)\right] \]

步骤2： 高斯积分计算：

\[ Z_0[J] = Z_0[0] \exp\left[-\frac{1}{2}\int d^4x d^4y J(x)D_F(x-y)J(y)\right] \]

步骤3： 两点关联函数：

\[ \langle 0|T\{\phi(x)\phi(y)\}|0\rangle = \left.\frac{1}{i^2}\frac{\delta^2 Z_0[J]}{\delta J(x)\delta J(y)}\right|_{J=0} = D_F(x-y) \]

第四章：重正化理论

12题

问题4.1 - 单圈重正化

题目描述

计算φ⁴理论中标量场质量的重正化常数 \(Z_m\) 到单圈阶。

解答

步骤1： 自能图贡献：

\[ -i\Sigma(p^2) = \frac{-i\lambda}{2} \int \frac{d^dk}{(2\pi)^d} \frac{i}{k^2-m^2+i\epsilon} \]

步骤2： 维度正规化计算：

\[ \Sigma(p^2) = \frac{\lambda m^2}{32\pi^2}\left[\frac{2}{\epsilon} + \text{有限项}\right], \quad \epsilon = 4-d \]

步骤3： 重正化常数：

\[ Z_m = 1 + \frac{\lambda}{32\pi^2\epsilon} \]
