第一章：声学基础

问题1.1 - 声速公式推导

题目描述

推导理想气体中的声速公式：

\[ c = \sqrt{\frac{\gamma p}{\rho}} \]

其中γ为绝热指数，p为压强，ρ为密度。

解答

步骤1： 考虑小振幅声波传播，气体状态变化为绝热过程：

\[ pV^\gamma = \text{常数} \]

步骤2： 线性化连续性方程和运动方程：

\[ \frac{\partial \rho'}{\partial t} + \rho_0 \nabla \cdot \mathbf{v} = 0 \]

\[ \rho_0 \frac{\partial \mathbf{v}}{\partial t} = -\nabla p' \]

步骤3： 绝热关系给出压强扰动与密度扰动关系：

\[ p' = \left(\frac{\partial p}{\partial \rho}\right)_s \rho' = c^2 \rho' \]

步骤4： 结合理想气体状态方程，得到声速公式：

\[ c = \sqrt{\frac{\gamma p_0}{\rho_0}} = \sqrt{\gamma R T / M} \]

第二章：声波传播

15题

问题2.1 - 平面声波的声强计算

题目描述

计算频率为1kHz、声压幅值为1Pa的平面声波在空气中的声强。已知空气特性阻抗为415Pa·s/m。

解答

步骤1： 声强定义：

\[ I = \frac{1}{T} \int_0^T p(t)v(t) dt \]

步骤2： 对于平面波，声压与质点速度同相：

\[ v(t) = \frac{p(t)}{\rho_0 c} \]

步骤3： 代入简谐声波表达式：

\[ p(t) = p_0 \cos(\omega t), \quad v(t) = \frac{p_0}{\rho_0 c} \cos(\omega t) \]

步骤4： 计算得到平均声强：

\[ I = \frac{p_0^2}{2\rho_0 c} = \frac{1^2}{2 \times 415} \approx 1.2 \times 10^{-3} \text{W/m}^2 \]
