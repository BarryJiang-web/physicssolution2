问题1.1 - 卢瑟福散射公式推导

题目描述

推导卢瑟福散射公式，证明散射截面为：

\[ \frac{d\sigma}{d\Omega} = \left(\frac{Ze^2}{16\pi\epsilon_0 E}\right)^2 \frac{1}{\sin^4(\theta/2)} \]

其中Z为靶核电荷数，E为入射粒子动能，θ为散射角。

解答

步骤1： 考虑库仑相互作用力：

\[ F = \frac{Ze^2}{4\pi\epsilon_0 r^2} \]

步骤2： 计算角动量守恒和能量守恒：

\[ b = \frac{Ze^2}{8\pi\epsilon_0 E} \cot(\theta/2) \]

其中b为碰撞参数

步骤3： 散射截面定义：

\[ \frac{d\sigma}{d\Omega} = \left|\frac{b}{\sin\theta}\frac{db}{d\theta}\right| \]

步骤4： 代入b的表达式得到卢瑟福公式：

\[ \frac{d\sigma}{d\Omega} = \left(\frac{Ze^2}{16\pi\epsilon_0 E}\right)^2 \frac{1}{\sin^4(\theta/2)} \]

第二章：氢原子光谱

12题

问题2.1 - 巴尔末系波长计算

题目描述

计算氢原子巴尔末系前三条谱线的波长（n=3,4,5→n=2）。

解答

步骤1： 里德伯公式：

\[ \frac{1}{\lambda} = R_H \left(\frac{1}{n_f^2} - \frac{1}{n_i^2}\right) \]

其中R_H=1.097×10⁷ m⁻¹为里德伯常数

步骤2： 计算Hα线(n=3→2)：

\[ \frac{1}{\lambda} = 1.097\times10^7 \left(\frac{1}{2^2} - \frac{1}{3^2}\right) \Rightarrow \lambda = 656.3 \text{nm} \]

步骤3： 计算Hβ线(n=4→2)：

\[ \lambda = 486.1 \text{nm} \]

步骤4： 计算Hγ线(n=5→2)：

\[ \lambda = 434.0 \text{nm} \]
